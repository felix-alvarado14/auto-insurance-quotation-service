# Auto Insurance Quotation Service

Microservicio REST orientado a gestionar el flujo inicial de cotizacion de seguros de automovil, preparado para crecer con una arquitectura limpia y modular.

## Stack tecnologico

- Next.js 15 (App Router)
- React
- TypeScript
- Tailwind CSS
- Prisma ORM
- MySQL
- Docker
- JWT
- Zod

## Roadmap

1. Configuracion inicial y estructura base del proyecto
2. Configuracion de infraestructura y entorno de desarrollo
3. Modelado de base de datos con Prisma y MySQL
4. Sistema de autenticacion y autorizacion con JWT
5. Desarrollo de modulos y servicios de negocio
6. Construccion de endpoints REST y validaciones
7. Pruebas, documentacion y despliegue
